# client

To install dependencies:

```bash
bun install
```

## Pre-reqs

### Bun

Read up on how to install Bun in their [official docs](https://bun.sh/docs/installation)

## Available scripts

### `bun dev`

Runs the dev env

### `bun test`

Runs the tests available in `src` dir

### `bun build`

Builds the app

### `bun lint`

Lints project
